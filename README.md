# EV Market Intelligence Dashboard

## About the Project
This Power BI dashboard was created to analyze Electric Vehicle market trends and provide meaningful insights through interactive visualizations.

The dashboard helps in understanding EV growth, vehicle distribution, manufacturer performance, pricing trends, and electric range analysis in a simple and visually appealing way.

---

## Tools Used
- Power BI
- Excel
- DAX
- Data Cleaning

---

## Dashboard Features
- Interactive Filters & Slicers
- KPI Cards
- State-wise Vehicle Analysis
- Manufacturer Comparison
- EV Type Distribution
- Model Year Trend Analysis
- MSRP & Electric Range Insights

---

## KPIs Included
- Total Brands
- Total Vehicles
- Average MSRP
- Average Electric Range

---

## Visualizations Used
- KPI Cards
- Map Visualization
- Line Chart
- Donut Chart
- Bar Chart
- Waterfall Chart
- Scatter Plot

---

## Key Insights
- Tesla contributed the highest number of EV vehicles
- Battery Electric Vehicles dominated the market
- EV adoption increased significantly over recent years
- Some states showed higher EV distribution compared to others
